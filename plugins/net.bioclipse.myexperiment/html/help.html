About myexperiment

Myexperiment is a website that make it easy to find and use scientific workflows and other research objects. It also allow you to build communities to share your workflow and research objects with others. You can read more about myexperiment on the website.

Myexperiment in Bioclipse

In Bioclipse you can search for and download workflows etc on myexperiment that is related to Bioclipse, i.e. the things been taged with "bioclipse". To be able to download e.g. a script from the site then you have to have its id-number. If you don't know the id-number you can get it either from the list that you get when writing myexperiment.list() in the JavaScrip console or by searching with for one or more words in the title of workflow. You may also click on this link to get a list (in your browser) of the items in myexperiment that has been taged with "bioclipse".

Myexperiment scripting

A short description of the availably commands for myexperiment in bioclipse is availably here and below is an example.
Let's say that we for some reason wants to know if there's some thing for Bioclipse on myexperiment that has something to do with OpenTox. So the start by writing this in JavaScript console:

> res = myexperiment.search("OpenTox")

When Bioclipse is done searching the result will be written in the JavaScript console, in this case it will say this:

List of size 8

This means that there is eight items that has OpenTox in its headline and the variable that we called res is the id of each of the items. If you want to download one of them e.g. the first in the list write:

> myexperiment.downloadWorkflow(res.get(0), "/MyExperiment/Opentox")

Except for downloading the item the line above also tells Bioclipse to save it as a txt-file with the name "OpenTox" in the director "MyExperiment" in your Bioclipse workspace.
